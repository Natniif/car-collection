## Getting Started 

### Running
Need to have a docker container open 
Run the command 
### `docker compose up --detach`

You can then start the web application from `localhost:1234`

### Tests
run tests from source using `phpunit test`

### Dependencies 
Unit tests depend on PHPUnit
